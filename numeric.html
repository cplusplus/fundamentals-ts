Function template randint

A separate per-thread engine of type default_random_engine (), initialized to an unpredictable state, shall be maintained for each thread.

template<class IntType> IntType randint(IntType a, IntType b); a ≤ b. If the template argument does not meet the requirements for IntType (), the program is ill-formed. A random integer i, a ≤ i ≤ b, produced from a thread-local instance of uniform_int_distribution<IntType> () invoked with the per-thread engine. void reseed(); void reseed(default_random_engine::result_type value); Let g be the per-thread engine. The first form sets g to an unpredictable state. The second form invokes g.seed(value). Subsequent calls to randint do not depend on values produced by g before calling reseed. reseed also resets any instances of uniform_int_distribution used by randint.
